基于 [supabase](https://supabase.com/) (Firebase开源替代品) 和 [Next.js](https://nextjs.org/) 实现简单博客系统。

## 启动

第一步进入项目根目录依次执行以下命令:

1. 安装依赖
```bash
pnpm install
```

2. 运行
```bash
pnpm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)
